Equipment Data
This window inputs the general data for assessment. The data definition may be used follows;

  • Equipment Number : The calling name or calling number of the equipment (not required).
  • Equipment Type : Type of the equipment such as Pipe components, Pressure Vessel and Atmospheric Storage Tank (required).
  • Methodology : The reference method which are used in this assessment (not required).
  • Unit : The assessment unit. This program provides 2 option 1) SI unit (mm, MPa and oC) and 2) UK unit (inch., psi and oF), (required).
  • Assessment Level : The level for assessment based on API 579-1/ASME FFS-1 2007 Fitness-For-Service. Level 1 assessment is more conservative than Level 2 assessment but also require less data than Level 2 assessment see Figure 1 (not required).
  • Analysis By : The name of the assessor (not required).
  • Analysis Date : Assessment date in DD/MM/Y (not required).
  • Equipment Image : Assessment picture or damaged picture of the equipment (not required).
