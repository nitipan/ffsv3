Design Data
This window inputs the design data of the equipment for assessment. The data definition may be used follows; Component Type: Type of the equipment such as Straight pipe, Cylindrical Shell, Spherical Shell, Hemispherical Head and Storage Tank (required).

  • Design Code : The standard or code which used for design the component (not required).
  • Design Pressure : The design pressure for the equipment (required).
  • Design Temperature : The design temperature for the equipment (required).
  • Inside Diameter : The original or design inside diameter (required).
  • Nominal Thickness, tnom : Previous metal loss associated with the governing thickness see Figure 2, (required).
  • Future Corrosion Allowance, FCA : Specified future corrosion allowance see Figure 2 (required).
  • Uniform Metal Loss, LOSS : Metal loss in the equipment prior to the assessment equal to the nominal minus the measured minimum thickness at the time of the inspection see Figure 2 (required).



  • Min. Required Longitudinal Thickness : Minimum required thickness based on the longitudinal membrane stress for a cylinder or cone, as applicable (required).
  • Min. Required Circumferential Thickness : minimum required thickness based on the circumferential membrane stress for a cylinder or cone, as applicable (required).
  • Weld joint efficiency : weld joint efficiency or quality factor from the original construction code, if unknown use 0.7.
  • Allowable RSF : Allowable remaining strength factor which is defined as

    RSF=LDC /LUC
    When LDC   is limit or plastic collapse load of the damaged component (component with flaws) and LUC   is limit or plastic collapse load of the undamaged component (required).
